# Video-games-sales-analysis-


I analyse a video games sales dataset (https://www.kaggle.com/datasets/lamskdna/video-games-sales). I use pyhton and MySQL to do some basic analyses for the dataset. This project is about learning the basics of SQL and database analyse tools. I visualize the data with matplotlib and Power Bi.
